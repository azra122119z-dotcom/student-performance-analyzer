# 🎓 Student Performance Analyzer

## 📌 Overview
A Python program that analyzes student marks and provides key insights like average, highest, and lowest scores, along with performance evaluation.

---

## 🚀 Features
- Dynamic subject input  
- Average, highest, and lowest score calculation  
- Performance evaluation  

---

## 🛠️ Tech
- Python

- Lists for data storage

- `if-elif-else` logic
